Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the first layer to making a website, similar to how foundation is the first layer to a house.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border indicates how much of a border we want around the outside of an element.
  • A box shadow adds a shadow around the outside of the frame.
  • When assigning text, you have the ability to use text-align to make it on the left, right, or centered.
  • You can also alter all of these factors to an image using img.
  • Using each variable per text in HTML, you can separate that out for assigning colors, spaces, display, etc. in CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls changes from the main branch to make sure you aren't pushing out of date information
  • git push origin branch-name: pushes new changes in the branch named to GitHub for verification and to alter main
